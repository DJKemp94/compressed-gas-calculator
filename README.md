# Compressed Gas Calculator

A lightweight calculator project for compressed gas scenarios, including supporting methodology notes, worked examples, and SVG assets.

## Contents

- `Feb_2026_Refresh_enhanced.html`: Main calculator interface.
- `Calculator_Methodology.md`: Notes on approach and assumptions.
- `Worked Examples.docx`: Example calculations.
- `*.svg`: Diagram/asset files used by the calculator.
- `Archive/`: Historical versions and backups.

## Usage

1. Open `Feb_2026_Refresh_enhanced.html` in a modern web browser.
2. Enter scenario inputs in the calculator UI.
3. Review outputs and cross-check with `Calculator_Methodology.md` and worked examples.

## Development Notes

- This repository currently uses static files (HTML/SVG/Markdown/Docx) and a small helper script (`add_scenario8.py`).
- No build step is required.
